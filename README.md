# FairyTales.V

## Emerging Technology Frameworks - Frontend Web Development Project

This project is a responsive frontend website for a fictional bookstore called FairyTales. It was created as part of the Emerging Technology Frameworks module at university to demonstrate practical frontend development skills using modern web technologies.

## Project Overview
The website presents a polished online bookstore landing page with sections for a featured book, author information, related books, and book details. The design focuses on creating an attractive and user-friendly interface for a digital reading experience.

## Purpose of the Project
This project was developed to:
- practice building a complete frontend web page
- apply responsive web design principles
- use Bootstrap components and custom CSS styling
- improve UI/UX design skills for web applications
- present a professional static website suitable for academic submission

## Features
- Responsive navigation bar
- Hero section with featured book details
- Author profile section
- Related books cards
- Styled tables for book information and offers
- Modern colors, hover effects, and button styling
- Clean and attractive layout for a bookstore website

## Tech Stack
- HTML5 for page structure
- CSS3 for custom styling and layout design
- Bootstrap 5 for responsive layout and UI components
- Local image assets for the bookstore visuals
- Git and GitHub for version control and project hosting

## Project Structure
- page.html - main HTML content of the website
- style.css - custom CSS styling and visual design
- image files - book covers and author images used in the page
- README.md - project documentation

## How to Run the Project
You can view the website locally in two simple ways:

1. Open the page.html file directly in your browser.
2. Or run a local server:

```bash
python -m http.server 8000
```

Then open:

```text
http://localhost:8000/page.html
```
